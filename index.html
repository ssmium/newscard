⚠️ 네이버 뉴스 검색을 사용하려면 Cloudflare Worker 설정이 필요해요
  1. 👉 workers.cloudflare.com 접속 (무료 계정 OK)
  2. Create Worker 클릭 → 함께 제공된 naver_proxy_worker.js 코드 전체 붙여넣기
  3. Save and Deploy → 생성된 URL 복사
    예: https://naver-proxy.이름.workers.dev
  4. 이 HTML 파일을 메모장/VS Code로 열어서
    PROXY_BASE = 'YOUR_WORKER_URL_HERE' 부분에 복사한 URL 붙여넣기
  5. 저장 후 GitHub Pages에 다시 올리면 검색이 돼요 ✓
뷰티·패션 리서치 — 키워드 선택 + 직접 추가
